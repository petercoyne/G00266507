Technical Details HTTP API Failte Ireland

The app queries one of three Failte Ireland API endpoints, depending on the type of resource requested.

 Map library Leaflet

Leaflet is an open source javascript library for mobile-friendly interactive maps.

 Ionic Native Plugin Geolocation

The app uses the geolocation plugin to locate users on the map.

 Additional style system TailwindCSS

In addition to ionic components, Tailwind was used to create some custom layouts.

 Misc Other Details

The app was built using the Ionic CLI, using Angular as the framework, and attempts to follow the submission guidelines.

It additionally integrates a MapBox map using the Leaflet javascript library, which displays results from a Failte Ireland API. The API functionality is housed in an Angular service. Placing markers and flying to different locations is handled in typescript or javascript using the Leaflet API. Data flows unidirectionally from the list to the map... it could be further improved by adding click bindings to the map markers, so that the list would update when a marker was tapped.

Tailwind CSS was used in places as I find it easier to work with than regular CSS/SCSS. It may appear to clutter the markup but I think the tradeoff is worth it.
